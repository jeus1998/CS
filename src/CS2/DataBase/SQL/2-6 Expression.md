
# Expression

### CASE - WHEN - ELSE - END

MySQL 기준 CASE문 사용방법 
```sql
SELECT CUSTOMERID, 
	   CASE CUSTOMERSEX
       WHEN 0 THEN 'MAN'
       WHEN 1 THEN 'WOMAN'
       ELSE 'NO SEX'
       END AS 성별, CUSTOMERNAME 
FROM CUSTOMER;
```
- ELSE 문 생략 가능 

CASE문 사용법2
```sql
SELECT CUSTOMERID,
	   CASE 
		   WHEN CUSTOMERAGE < 20 THEN '미성년자'
		   WHEN CUSTOMERAGE >= 20 THEN '성인'
       END AS AGE_GROUP, CUSTOMERAGE, CUSTOMERSEX, CUSTOMERNAME
FROM CUSTOMER;
```
